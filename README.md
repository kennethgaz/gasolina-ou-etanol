# Gasolina ou Etanol?
Esse aplicativo faz o cálculo para ver se compensa mais utilizar gasolina ou etanol.

# Criar um novo app
```
npm install -g react-native-cli
react-native init newApp
```

# Ver os aparelhos conectados via USB
```
adb devices
```

# Iniciar a aplicação
```
react-native run-android
```
